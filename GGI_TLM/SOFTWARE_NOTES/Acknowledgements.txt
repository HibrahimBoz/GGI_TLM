The GGI_TLM software made available here has been developed within the following projects:

EPSRC/ BAE SYSTEMS Flaviir project
EU Framework7 HIRF-SE project

The software uses the EISPACK library, the source code of which is included in GGI_TLM.

The seeding of the random number generator (read_random_number_seed.F90) is based on a 
fortran90 subroutine from the webpage:

http://gcc.gnu.org/onlinedocs/gfortran/RANDOM_005fSEED.html

The Python script used in the automatic creation of simulation documents uses the code 

PyRTF (Version 0.45) by Simon Cusack, available under the gpl license
License: http://www.gnu.org/licenses/gpl.html
